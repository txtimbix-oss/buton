# Бутон — цветочная мастерская (Санкт-Петербург)

Многостраничный сайт цветочного магазина «Бутон»: витрина, каталог, карточка
товара, конструктор букета, оформление и отслеживание заказа, подписка и
подарочные карты.

## Технологии

- Статические HTML-страницы + общий стиль `buton.css`.
- Интерактив на **React 18** (UMD) + **ReactDOM**, JSX компилируется в браузере
  через **Babel Standalone** — всё подключается с CDN (unpkg), сборка не нужна.
- `shell.js` подставляет общую шапку/плашку/футер на статические страницы
  (где нет своего React-приложения).

## Как запустить локально

JSX подгружается через `fetch`, поэтому открытие по `file://` упрётся в CORS.
Нужен любой локальный статический сервер из папки проекта:

```bash
# Python 3
python -m http.server 8000
```

Затем открыть в браузере: <http://localhost:8000/Витрина.html>

## Структура

- `*.html` — страницы
- `*.jsx` — React-приложения страниц
- `buton.css` — общие стили и дизайн-система (цвета, кнопки, карточки)
- `shell.js` — общая обёртка для статических страниц
- `cabinet/` — раздел личного кабинета
- `screenshots/`, `uploads/` — артефакты разработки и исходники (не в git)
